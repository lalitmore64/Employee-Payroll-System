# Employee Payroll System (Core Java + OOP)

A console-based Employee Payroll System built using **Core Java** and **Object-Oriented Programming (OOP)** concepts.

## Features

* Add Full-Time and Part-Time Employees
* Calculate salary based on employee type
* Demonstrates OOP concepts:

  * Abstraction
  * Inheritance
  * Polymorphism
  * Encapsulation

##  Technologies Used

* Core Java
* OOP Principles
